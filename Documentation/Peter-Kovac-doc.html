Záverečná správa

Mojou úlohou v projekte bolo vybrať vhodnú databázu pre uchovávanie XML dokumentov a naimplementovať rozhrania, ktoré by riešili prácu s jednotlivými typmi dokumentov.

Výber XML databáze

K uchovávaniu xml dokumentov som nakoniec zvolil databázu BaseX.

Rozhrania pre prácu s databázou

Na začiatku sme si v tíme stanovili, akú funkcionalitu má spĺňať naša aplikácia a popísali sme ju v UML diagrame. Podľa tohto diagramu som vytvoril jednotlivé API pre každý typ dokumentu obsahujúce funkcie pre prácu s databázou. Vyhľadávanie dát som riešil pomocou Xquery.
